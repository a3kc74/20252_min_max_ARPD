# Metaheuristic slides for combinatorial optimization

Downloaded public slide/PDF materials related to metaheuristics, local search, and combinatorial optimization.

## Files

1. `HUST_Chuong3_Tim_kiem_Heuristic.pdf`
   - Source: https://users.soict.hust.edu.vn/huonglt/AI/Chuong%203.%20Tim%20kiem%20Heuristic.pdf
   - Origin: HUST/SoICT course material.
   - Relevance: heuristic search foundations, useful for explaining local search and search-space exploration.

2. `MiniCP_06_Routing_Optimization_LNS.pdf`
   - Source: https://minicp.github.io/website/_downloads/c8f821e8a8eb39f376577befbe663de7/06-circuit-optim-lns.pdf
   - Origin: MiniCP teaching material.
   - Relevance: routing optimization and large neighborhood search.

3. `MiniCP_09_Black_box_Search.pdf`
   - Source: https://minicp.github.io/website/_downloads/bb299ebcdbc40fb687161c55df4e575d/09-black-box-search.pdf
   - Origin: MiniCP teaching material.
   - Relevance: local search, simulated annealing, tabu search, and metaheuristic black-box search.

4. `Rosati_RL_Multi_Neighborhood_Local_Search_IFORS23.pdf`
   - Source: https://www.robertomariarosati.eu/publication/learning-4-local-search/Rosati-Pres_IFORS23.pdf
   - Origin: IFORS 2023 presentation by Roberto Maria Rosati.
   - Relevance: multi-neighborhood local search for combinatorial optimization.

5. `EURO2025_Metaheuristic_Algorithms_Pisinger.pdf`
   - Source: https://backend.orbit.dtu.dk/ws/portalfiles/portal/412135882/EURO-2025-Pisinger.pdf
   - Origin: EURO 2025 presentation by David Pisinger.
   - Relevance: modern metaheuristic algorithm overview.

## Notes

Priority was given to HUST materials where publicly accessible. I found one direct HUST/SoICT PDF matching heuristic search. The remaining files are public international slide decks that are closely related to the thesis topic: local search, large neighborhood search, routing, and metaheuristics for combinatorial optimization.

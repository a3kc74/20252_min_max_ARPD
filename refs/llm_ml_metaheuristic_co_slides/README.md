# Slides: LLM / ML / metaheuristics for combinatorial optimization

This folder contains public slide decks related to representative works on using large language models, machine learning, and metaheuristics for combinatorial optimization.

## Downloaded files

1. `NeurIPS2025_LLM_End_to_End_CO_Solvers.pdf`
   - Source: https://neurips.cc/virtual/2025/loc/san-diego/poster/115835
   - Direct PDF: https://neurips.cc/media/neurips-2025/Slides/115835.pdf
   - Topic: large language models as end-to-end combinatorial optimization solvers.
   - Relevance: directly about LLM-based CO solving.

2. `ICLR2025_LLM_Neural_Solvers_for_VRP.pdf`
   - Source: https://iclr.cc/virtual/2025/poster/33144
   - Direct PDF: https://iclr.cc/media/iclr-2025/Slides/33144.pdf
   - Topic: LLM-powered neural combinatorial optimization solvers for vehicle routing.
   - Relevance: LLM + neural solver + routing/CO; especially close to routing problems.

3. `ICML2025_MCTS_LLM_Automatic_Heuristic_Design.pdf`
   - Source: https://icml.cc/virtual/2025/poster/45984
   - Direct PDF: https://icml.cc/media/icml-2025/Slides/45984_i4beMa3.pdf
   - Topic: Monte Carlo tree search with LLMs for automatic heuristic design.
   - Relevance: LLM-guided heuristic discovery for combinatorial optimization.

4. `LANL_Lodi_ML_for_CO_Methodological_Tour.pdf`
   - Source: https://cnls.lanl.gov/External/gsabstracts2021/Lodi.pdf
   - Topic: machine learning for combinatorial optimization, methodological overview.
   - Relevance: classic ML4CO survey-style material by leading researchers in the field.


5. `LLM4AD_Material.pdf`
   - Source: https://www.cs.cityu.edu.hk/~zhichalu/assets/slides/LLM4AD_Material.pdf
   - Related project/page: LLM4AD / automated algorithm design with large language models.
   - Topic: LLM-based automated algorithm design, heuristic generation, and optimization.
   - Relevance: directly connected to LLM + metaheuristic / heuristic design for CO.
